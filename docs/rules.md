# ナショナルエコノミー ゲームルール

## コンポーネント

| 種類 | 数量 |
|------|------|
| 初期公共職場カード | 採石場×1、鉱山×1、学校×1、大工×(人数−1) |
| 建物カード | 48枚（25種類） |
| 労働者カード | 6枚 × 2セット |
| ラウンドカード | 9枚 |
| スタートプレイヤーカード | 1枚 |
| 消費財カード | 無限（効果なし・コスト支払い専用） |
| 未払い賃金カード | ∞（ペナルティ用、終了時−3点/枚） |

---

## セットアップ

### プレイヤーごとの準備

1. 各プレイヤーは色を決め、**労働者カード2枚**を受け取る
2. 自分の手元に建物置き場を確保する
3. スタートプレイヤーを決め、スタートプレイヤーカードを渡す
4. スタートプレイヤーから時計回りに**初期資金**をサプライから受け取る
   - 手番1番目：$5
   - 手番2番目：$6
   - 手番3番目：$7（3人プレイ時は最後）
   - 手番4番目：$8（4人プレイ時のみ）
   - ※2人・3人プレイも$5から始め1ドルずつ増加
5. 建物カードの山札から**3枚**を引いて手札とする

### 共有スペース（場）の準備

1. **初期公共職場**を表向きで中央に並べる
   - 採石場：1枚
   - 鉱山：1枚
   - 学校：1枚
   - 大工：プレイ人数−1枚（4人→3枚、3人→2枚、2人→1枚）
2. **建物カード**をよく混ぜて裏向きの山札にし、隣に捨て札置き場を作る
3. **ラウンドカード**を裏向きに1〜9の順に重ねて置く
4. **消費財カード**を重ねて山にする

### お金の管理場所

- **サプライ**：お金カード・未払い賃金カード・残りの労働者カードを置く場所
- **家計**：サプライとは別の場所（最初は空の状態）
  - プレイヤーが受け取った初期資金は各自の家計に置く

## ゲームの流れ

### ラウンド構造
- 全9ラウンド制
- スタートプレイヤーから時計回りに手番を行う
- **全員の労働者がすべて配置されたらラウンド終了**

### 手番でできること

自分の手番が来たら、手元の労働者カード1枚を職場に置き、その効果を得る。全員の労働者がなくなるまで繰り返す。

### 職場のルール

- 1つの職場には**1ラウンドに1人の労働者**しか置けない
- **例外：鉱山**は複数の労働者を置ける（同一プレイヤーが複数回利用も可）
- 建設した建物は**所有者専用**の職場になり、他プレイヤーは使用不可
- 一部の建物（倉庫・法律事務所など）は職場として使えないが、**持っているだけで効果を発揮する「施設」**として機能する

### 建物の建設

「建物を作る」機能を持つ職場（大工・建設会社・ゼネコン・二胡市建設）に労働者を置いて行う：

1. 手札から建てたい建物カードを選ぶ
2. **コスト（左上の数値）分だけ手札の他のカードを捨てる**（建物カード・消費財カードどちらでも可）
   - 捨てた建物カード → 捨て札置き場（表向き）
   - 捨てた消費財カード → 消費財の山に戻す
3. 建設した建物を自分の場に表向きで置く（以降、自分専用の職場になる）

> ※ コストはお金ではなく**手札のカードを捨てる**ことで支払う
> ※ 建物の効果は**必ず全て適用**すること。一部でも実行できない場合は使用不可

**建設関連の特殊建物：**
- **建設会社**：コストが1安くなる大工として機能
- **ゼネコン**：建設後に山札から2枚引ける
- **二胡市建設**：同コストの建物を2つ同時に建設（1つ分のコストのみ支払う）

### 建物カード山札が切れた場合

1. 捨て札を切り直して新しい山札を作る
2. 捨て札も尽きた場合は、不足分を**消費財カード**で代替する

### ラウンド終了処理

1. **賃金の支払い**：労働者数 × ラウンドカードの賃金額 を家計からサプライへ支払う
   - 研修中の労働者も賃金が発生する
2. **家計が不足している場合**（以下の順で対処）：
   - **① 建物を売却**：売却不可でない建物を選んで売却し、公共職場へ移す。売却額 = カード下部中央の**資産価値**（左上のコストではない）と同額をサプライから受け取る
     - 不足額を補えるなら**最低限の売却のみ**可。必要以上に売ることはできない
     - 例）$20不足・所有建物が$10/$18/$20の場合：$20のみ売却は○、$10+$18は○、$10+$20は×（過剰）
   - **② 未払い賃金カードを受け取る**：売却可能な建物をすべて手放してもまだ不足している場合、支払える分だけ支払い、不足額$1につき未払い賃金カードを1枚受け取る
3. **次のラウンドカード**をめくり、記載の公共職場を場に追加する
4. スタートプレイヤーを次の人に移す

## 初期公共職場の効果

| 職場 | 効果 |
|------|------|
| 採石場 | 建物カード山札の一番上を1枚引く。スタートプレイヤーになる（すでに持っている場合は移動しない） |
| 鉱山 | 建物カードを1枚引く。複数の労働者を置ける唯一の職場（同一プレイヤー複数回利用も可） |
| 学校 | 労働者を1人増やす。ストックから自分の色の労働者を「研修中（裏向き）」で手元に置く。翌ラウンドから使用可能 |
| 大工 | 手札の建物カードを1枚建設する（コストは手札を捨てて支払う） |

---

## 家計とサプライ

- **家計**：各プレイヤー個人のお金
- **サプライ**：場の共有財源（固定総量・ゲーム内を循環）
- 賃金を支払うと → サプライへ
- 効果でお金を得るとき → サプライから引き出す
- **サプライが空の場合**：誰もお金を受け取れない

## ラウンドカード一覧

各ラウンドの開始時にそのラウンドのカードをめくり、記載の公共職場を場に追加する。

| ラウンド | 賃金 | 追加される公共職場 | 職場の効果 |
|---------|------|----------------|----------|
| 1 | $2 | 採石場・学校・大工・鉱山 | 初期公共職場を参照 |
| 2 | $2 | 露店 | 手札1枚捨てて家計から$6獲得 |
| 3 | $3 | 市場 | 手札2枚捨てて家計から$12獲得 |
| 4 | $3 | 高等学校 | 労働者数を4人になるよう増やす（すでに4人以上なら使用不可） |
| 5 | $3 | スーパーマーケット | 手札3枚捨てて家計から$18獲得 |
| 6 | $4 | 大学 | 労働者数を5人になるよう増やす（すでに5人以上なら使用不可） |
| 7 | $4 | 百貨店 | 手札4枚捨てて家計から$24獲得 |
| 8 | $5 | 専門学校 | 労働者を1人追加（研修中にならず即戦力。そのラウンドから配置可能。賃金はそのラウンド終了時から発生） |
| 9 | $5 | 万博 | 手札5枚捨てて家計から$30獲得 |

> ※ 商店系（露店・市場・スーパーマーケット・百貨店・万博）で捨てるカードは建物カード・消費財カードどちらでも可（確認済み）

## ゲーム終了（9ラウンド後）

**最終得点 = 手元の建物の資産価値合計 + 残りの家計のお金 − 未払い賃金カード枚数 × 3点**

最高得点のプレイヤーが勝者。
